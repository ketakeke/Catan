# 🏝️ カタン練習アプリ (Catan Practice App)

ブラウザだけで遊べるカタン（基本ルール）の完全再現ゲームです。  
CPU対戦でカタンの練習ができます。

## 🎮 プレイ方法

`catan.html` をブラウザで開くだけ。サーバー不要。

## ✨ 機能一覧

| 機能 | 説明 |
|------|------|
| ボード生成 | 19ヘックスのランダム配置（6/8隣接回避） |
| 初期配置 | ラウンド1→2（逆順）、2回目で資源獲得 |
| サイコロ | 完全ランダム2D6（CPU強さに依存しない） |
| 資源分配 | 開拓地=1、都市=2 |
| 建設 | 街道・開拓地・都市・発展カード |
| 距離ルール | 開拓地の隣接配置禁止 |
| 盗賊 | 7出目 → 半分廃棄 → 盗賊移動 → 強奪 |
| 発展カード | 騎士・勝利点・街道建設・収穫・独占（全5種） |
| 銀行交易 | 4:1 / 港3:1 / 専門港2:1 |
| 最長交易路 | DFS計算、5本以上で+2VP |
| 最大騎士力 | 3枚以上で+2VP |
| 勝利条件 | 10VP達成で即勝利 |
| CPU AI | 5段階の難易度設定 |
| プレイヤー | あなた + CPU×3（4人固定） |

## 🤖 CPU難易度

| Lv | 名称 | 特徴 |
|----|------|------|
| ⭐1 | 初心者 | ほぼランダムに行動 |
| ⭐2 | 見習い | 基本的な戦略あり |
| ⭐3 | 中級者 | バランスの取れたプレイ |
| ⭐4 | 上級者 | 効率重視、資源バランス考慮 |
| ⭐5 | 達人 | 最適手を追求、拡大戦略 |

## 🛠 技術仕様

- **単一HTMLファイル** — 外部依存なし
- **HTML5 Canvas** でボード描画
- **Vanilla JS** — フレームワーク不使用
- レスポンシブ対応（PC / タブレット）
